# cocode.dk

Personal brand website for Babak Bandpey — AI agent development, workflow automation, and cybersecurity compliance for Danish SMEs.

## Tech Stack

- Vanilla JavaScript SPA (no framework)
- Webpack 5 + Babel
- Warm Glassmorphism design system
- 2 languages: English, Danish

## Development

```bash
npm install
npm run dev          # Dev server on localhost:8080
npm run build        # Production build → /dist
npm test             # Run tests
npm run lint:length  # Check 200-line file limit
```

## Deployment

Pushes to `main` deploy automatically to GitHub Pages via GitHub Actions.
