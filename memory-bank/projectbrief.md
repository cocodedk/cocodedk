# Project Brief: cocode.dk

## Overview
cocode.dk is a freelance IT consultancy website for showcasing services and expertise. It features an interactive network graph visualization that demonstrates the interconnections between various services and technologies offered.

## Core Requirements
- Interactive network graph visualization as the main interface
- Multilingual support (11 languages)
- Node-based navigation for exploring services
- Dark-themed, high-contrast design for readability
- Responsive layout that adapts to different screen sizes

## Primary Goals
- Showcase IT consultancy services including software development and cybersecurity
- Demonstrate technical expertise through the visualization itself
- Provide clear information about service offerings
- Enable visitors to easily contact for consulting services

## Technical Specifications
- Built with vanilla HTML, CSS, and JavaScript
- Canvas-based visualization
- No reliance on external libraries for core functionality
- Simple and maintainable codebase
