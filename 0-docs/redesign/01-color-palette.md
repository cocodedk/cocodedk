# Warm Color Palette

## Primary Colors (Backgrounds & Accents)
- **Deep Charcoal**: `#1a1a1a` - Main background
- **Dark Burgundy**: `#3d1a2a` - Secondary background, gradient stops
- **Deep Rust**: `#5c2e1f` - Tertiary background layer

## Warm Accents (Text & Borders)
- **Warm Coral**: `#e8735e` - Primary accent, hover states
- **Amber Gold**: `#d4a574` - Secondary accent, links
- **Warm Orange**: `#f39c12` - Highlights, active states
- **Terracotta**: `#b8532d` - Card borders, tertiary accents

## Glass Effects Base
- **Glass Light**: `rgba(255, 255, 255, 0.1)` - Frosted glass base
- **Glass Medium**: `rgba(255, 255, 255, 0.05)` - Subtle glass
- **Shadow Dark**: `rgba(0, 0, 0, 0.4)` - Deep shadows

## Text Colors
- **Primary Text**: `#ffffff` - Main text
- **Secondary Text**: `#e0e0e0` - Muted text
- **Tertiary Text**: `#a0a0a0` - Inactive text

## Usage Guidelines
- **Backgrounds**: Deep Charcoal + Burgundy gradients
- **Buttons & CTAs**: Warm Coral
- **Card Borders**: Terracotta glows
- **Links**: Amber Gold
- **Highlights**: Warm Orange
- **Text**: White with opacity variations

## Contrast Ratios (WCAG AA Compliance)
- White on Deep Charcoal: 18:1 ✓
- Warm Coral on Deep Charcoal: 5.2:1 ✓
- Amber Gold on Deep Charcoal: 4.8:1 ✓
