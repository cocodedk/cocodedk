# cocode.dk Logo & Branding

## Logo Design
Saturn-inspired planet with rings representing "cocode.dk"

**Concept:**
- Planet Saturn symbolizes exploration, innovation, and cosmic reach
- Orbital rings represent connectivity and system architecture
- Warm coral/amber gradient reflects the site's glassmorphism theme
- Code brackets `< />` accent reinforces tech identity

## Files

## Color Palette
- Primary: #FF8A5B (Coral orange)
- Secondary: #E85A3F (Dark coral)
- Accent: #FFB88C (Light amber)
- Background: #1a1a2e (Dark navy)

## Usage
- Full logo: Marketing materials, headers, about pages
- Favicon: Browser tabs, bookmarks, PWA icons
- Maintains recognition at all sizes (16px to 200px+)
